# Sistema-libros-electronicos

## Descripción 
Planificacion del proyecto de sistema de libros electronicos desarrollado en Go aplicando el uso de programación funcional.
Permite la gestión de libros, gestion de usuarios y procesos de compra o descarga.

## Funcionalidades
- Registro de libros
- Registro de usuarios
- Listado y búsqueda de libros
- Gestión de compras y descargas

## Tecnologías
- Lenguaje: Go
- Programación Funcional
- Editor: Visual Studio Code
